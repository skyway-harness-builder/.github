# Getting help

- **Bug reports & feature requests** — [skyway-harness-builder/issues](https://github.com/skyway-harness-builder/issues), the central tracker for the whole product. Please include `skyway --version` output and, for run problems, the run id from `skyway logs`.
- **Questions about a specific workflow** — same tracker, with the `workflow-library` label.
- **Security problems** — never a public issue; see [SECURITY.md](SECURITY.md).
- **Everything else** — [skylence.be](https://skylence.be).

Before filing: `skyway run check-sky-environment --var dir=.sky/workflows` diagnoses the most common setup problems (missing CLI dependencies, auth, unset secrets) in seconds.
